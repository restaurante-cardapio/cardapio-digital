# 🍽️ Cardápio Digital com Painel Administrativo

Um sistema completo de cardápio digital para restaurantes, com foco em simplicidade, integração com WhatsApp e gestão autônoma via painel administrativo.

## ✨ Funcionalidades

### 🛒 Para o Cliente (Interface Principal)
- **Navegação Intuitiva:** Categorias organizadas e busca de produtos em tempo real.
- **Customização de Pedidos:** Adicionais e observações integradas.
- **Checkout em Etapas:** Identificação, Upsell de acompanhamentos e bebidas, e revisão final.
- **Integração WhatsApp:** Envio do pedido formatado diretamente para o restaurante.
- **Clube VIP e Fidelidade:** Sistema de pontos e captura de leads para marketing.
- **Temas:** Suporte a modo claro e escuro.

### 🛠️ Para o Restaurante (Painel Admin)
- **Gestão de Produtos:** Cadastro, edição, controle de estoque e pausa de vendas.
- **Configurações da Loja:** Alteração de logo, banners, cores, taxas de entrega e horários de funcionamento.
- **Relatórios:** Visão geral de vendas do dia e faturamento.
- **Marketing:** Lista de leads (clientes VIP) e ferramentas de indicação.

## 🚀 Como usar

1. Clone o repositório.
2. Abra o arquivo `index.html` e use o botão "Entrar / Cadastrar" no topo para criar sua conta.
3. Após o login como restaurante, você será redirecionado para o Painel Admin.
4. O link para os clientes acessarem o cardápio será exibido no dashboard do painel.

## 🛠️ Tecnologias

- HTML5 / CSS3 (Variáveis CSS, Flexbox, Grid)
- JavaScript Vanilla (ES6+)
- LocalStorage (Para persistência de dados sem necessidade de banco de dados externo)
